# Actix-web: Rust web framework

> 這個專案最主要用於學習 Rust 語言，並且實作一個簡單的 Web 框架。

## 使用到的套件介紹:

- `actix-web`: 一個 Rust 的 Web 框架。
- `actix-cors`: 一個 actix-web 的 CORS 中間件。
- `log`: 一個 Rust 的日誌框架。
- `env_logger`: 一個 log 的環境變數日誌輸出器。
- `serde`: 一個 Rust 的序列化/反序列化框架。 (features = ["derive"])
